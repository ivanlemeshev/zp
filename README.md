## Требования

* PHP >= 7.1

## Установка

```bash
$ git clone
$ cd zp
$ composer install
```

## Запуск приложения

```bash
$ composer start
```

Приложение доступно по адресу [http://localhost:8090](http://localhost:8090) 

## Отсановка приложения

```bash
$ composer stop
```

## Запуск тестов

```bash
$ composer start
$ composer test
```
